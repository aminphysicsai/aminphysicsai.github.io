How to Upload New Articles

A complete step-by-step guide to publishing new content on your Amin Physics AI GitHub Pages website

Step-by-Step Guide to Upload New Articles

1 Prepare Your Article Content

Create your article in HTML format. You can use any text editor (like VS Code, Notepad++, or even Notepad) to create an HTML file.

Recommended Article Structure:

  • Create a folder for your article (e.g., "quantum-mechanics-basics")
  • Inside this folder, create an index.html file
  • Create a subfolder for images (e.g., "images") if needed
  • Your HTML should include proper meta tags, title, and structured content

2 Connect to Your GitHub Repository

You need to have Git installed on your computer. Open your terminal or command prompt and navigate to your local repository folder.

# Navigate to your local repository
cd /path/to/aminphysicsai.github.io

# Check the status of your repository
git status

# If you haven't cloned the repository yet:
git clone https://github.com/aminphysicsai/aminphysicsai.github.io.git
cd aminphysicsai.github.io

3 Add Your New Article Files

Copy your article folder (with all HTML, CSS, and image files) to the appropriate location in your repository.

# For example, if you want to add a "quantum-physics" article:
# Copy your article folder to the repository
# Then add it to Git:
git add quantum-physics/

# Or add all new files:
git add .

# Check what you've added:
git status

4 Commit and Push Your Changes

Commit your changes with a descriptive message and push them to GitHub.

# Commit your changes
git commit -m "Added new article: Introduction to Quantum Physics"

# Push to GitHub
git push origin main

Note: If you're using the default branch master instead of main, use git push origin master instead.

5 Update the Main Navigation

To make your article accessible from the main website, update the navigation menu in your main HTML files.

<!-- In your index.html or main navigation file -->
<li><a href="quantum-physics/">Quantum Physics</a></li>
<li><a href="relativity/">Theory of Relativity</a></li>
<!-- Add your new article link here -->
<li><a href="new-article-folder/">Your New Article</a></li>

6 Verify Your Publication

After pushing your changes, visit your website to verify the article is live:

Website URL: https://aminphysicsai.github.io/new-article-folder/

GitHub Pages may take a few minutes (up to 10 minutes) to deploy your changes.

Important Tips

  • Keep file and folder names lowercase with hyphens (no spaces)
  • Optimize images for web (use .jpg or .webp format)
  • Test your article locally before pushing to GitHub
  • Use semantic HTML tags for better structure
  • Include meta descriptions for SEO

Next Steps

Once your article is live, consider:

  • Sharing on social media
  • Linking from your homepage
  • Creating a sitemap update
  • Adding to your article index page
